About Jump Start

Pantheon's Jump Start program is designed for agencies to standardize on development best practices such as git based deployment, a dev/test/live workflow, and parallel feature branch development with Multidev. With Jump Start, your team will be fully up and running on Pantheon in just a couple days of time.
